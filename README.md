# dfad MCP Server

このプロジェクトは、AIコーディングエージェント（Claude CodeやCursor、Antigravityなど）に**Android端末（実機・エミュレータ）を自律的に操作・検証させるためのMCP(Model Context Protocol) サーバー**です。

システム上の `adb` コマンドを内部でラップしており、AIエージェント自らが現在のAndroid画面のスクリーンショットやUIのツリー構造（View Hierarchy）を取得し、解析した座標をもとにタップやテキスト入力などの操作を行えるようにします。

## 主な機能

AIエージェントに以下のネイティブツール（能力）を提供します：

| ツール名 | 説明 |
| :--- | :--- |
| `get_screen_state` | 現在の画面のスクリーンショット(Base64画像)とUI構造(XMLダンプ)を一緒にAIに送信します。画面状態の認識に使用します。 |
| `tap` | 画面の指定された座標 (X, Y) をタップします。 |
| `input_text` | フォーカスされている入力欄にテキストを入力します。 |
| `key_event` | ホームボタン(3)、戻るボタン(4)、エンターキー(66)などのシステムキーを発行します。 |
| `launch_app` | 指定したパッケージ名のAndroidアプリを起動します。 |

## システム要件

1. **Node.js**: v18以上を推奨
2. **Android SDK Platform-Tools (`adb`)**: ターミナル上でグローバルに `adb` コマンドが実行可能であり、対象のAndroid端末やエミュレータが認識されている（`adb devices` で表示される）こと。

## セットアップ方法

AIエージェント（ClaudeやCursor等）でこのMCPサーバーを使えるようにするために、**対話型のインストールスクリプト**を用意しています。

以下のコマンドを実行し、画面の指示に従ってご利用中のAIツールを選択してください。

```bash
# プロジェクトのディレクトリにて依存関係をインストール（初回のみ）
npm install

# 対話型インストールウィザードを起動
npm run install:mcp
```

### ウィザードの選択肢について
- **1. Claude Code (CLI)**: コマンド操作で完全に自動で追加されます。開発とテスト並行作業に最もおすすめです。
- **2. Claude Desktop (Mac)**: 設定ファイルを自動で上書きし追加します。（追加後アプリの再起動が必要です）
- **3. Cursor**: エディタのGUI上でどこに何を入力すべきかの案内をターミナルに表示します。
- **4. Antigravity**: AIチャット上でどのようにテスト開始を指示するべきかを表示します。
- **5. その他**: Windsurf等の他のMCPクライアント向けに設定用のJSONパラメータを表示します。

## 利用方法（プロンプトの例）

セットアップが完了したら、実際に対象のAIエージェントと会話を始めて、以下のように指示を出すだけで画面を見ながらの自律的なAndroidテストがスタートします。

> **AIへのプロンプト例:**  
> 「ターミナルで `./gradlew installDebug` を実行してアプリをインストールした後、あなたの `launch_app` ツールを使ってアプリを起動してください。起動したら現在の画面を `get_screen_state` で確認し、ログインボタンを見つけてタップしてログインフローの動作検証を行ってください。」

AIは画面全体のレイアウトやボタンの位置を視覚とXMLの両方から確実に判断し、自分で座標を計算して操作（自己修正ループ）を進めます。
